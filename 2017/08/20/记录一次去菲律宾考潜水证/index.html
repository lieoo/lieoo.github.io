记录一次去菲律宾考潜水证

发表于 2017-08-20

一直心心念念的潜水，这次终于有了时间出去玩，害怕自己对待事情只有三分钟热度，所以赶紧计划了行程。在挑选国家之间做出了很多选择，泰国的涛岛、菲律宾的萨邦还有马来西亚的沙巴，考虑到很多因素，也根据朋友的推荐，最后选择了菲律宾的萨邦岛。

  • 这是本人第一次出国也是自己单独一个人出行，从计划潜水到购买机票只用了两天时间，所以有些花费比较高，例如：机票 可以通过宿务航空的微信公众号查询机票，经常会有特价活动，广州往返马尼拉（菲律宾首都）的价格经常在700-800元就可以搞定，这次的花费就比较多，下次有了经验会花费更少一些，也是我这篇博客的目的之一。

出行路线

关于出行路线我是按照穷游App的一篇文章 菲律宾PG岛完整交通攻略（2017年5月更新水飞时间）
大概的路线与方向没有问题，可以参考此篇文章出行，只对于此篇文章进行详细补充和可能出现的异处截图说明。

到达菲律宾！

  1. 下了飞机第一件事情，要先出海关才可以拿到自己的行李，所以可以在排队等待出海关的时候，连接上机场的 Wifi，给家里报平安。
    拿到行李后，我是跟随人流多的地方走，也是穷游帖子上提到的机场门口，购买到了 Globe 运营商的电话卡与充值卡，按照帖子上面发送短信开通激活电话卡后充值金额和办理流量业务。

  2. 出机场门口打黄颜色的出租车或者 Uber 到 Jac Liner 或者 Jam Liner 或者 DLTB Liner，三个大巴站点在同一个地方，国外的 Uber 可以使用现金支付，不同于国内。

  3. 在车头会有牌子标记车辆所属运营公司与到达终点，看到有写 Batangas Port（八打雁港口）或者问一下附近的乘务员，他们很愿意帮助你，只管上车就OK了。
    我乘坐的就是 JAM 公司的车辆 有一些破 没办法

上车后，乘务员会走两次，第一圈是询问你到哪里，并且给你票，第二圈是付款以及找零。
乘务员并没有询问我，可能对于多数外国人来说都是去终点 Batangas Port，所以就没有询问。

图为车票，价格为 167 Peso（约23人民币），时长为2小时左右，全程高速
（坐车的时候就在想 为什么对比起来差距价格差距这么大…没有过路费…恩…）
4.到达 Batangas Port 终点站准备购买船票 160peso以及码头票 50peso（类似于门票），根据走路过去五分钟左右，期间大约有4-5个小黑询问需要帮忙购买船票之类的，都被我拒绝了。他们的服务都是需要给小费，对于金牛座的我来说，很近还是自己动手比较好😏。
以下三张图为帖子摘取，路线是正确的，我自己没有在这个地方拍照，很小的地方，很好找。



买票后，查看票上的时间等待上船即可。

下船后在出口处会有人主动向你收取环境保护税（ENVIRONMENTAL USERS FEE） 50peso。
也就是说从机场到达Batangas Port的价格为 167peso+160peso+50peso+50peso = 427peso，大约60元人民币。这样算下来，其实还是很便宜的。

到达潜水店!

下船后直接在微信上面联系了潜水店家，他们很友好的为我们接风，拿行李。因为第一天到达潜水店太晚，教练让我好好休息一晚，所以正式的学习其实是安排在到达菲律宾的第二天。
从教练口中得知教学课程为：
第一天：学习潜水理论，教练说明潜水可能造成的危害，签署免责声明，观看潜水视频以及背题
第二天：平静水域（泳池）学习20多个水下动作机票，学习潜水理论，背题
第三天：开放水域（大海）复习几个在平静水域动作，潜水表的使用
第四天：Fun dive (FD) 自由潜水一次，期末考试

傍晚时候出去闲逛，晚上的萨邦岛非常热闹，很多酒吧，很多欧洲白人，很酷。
看到一个摊子，卖着超级无敌加长型面包，差不多7块钱人民币，一天没吃东西，很饿，没忍住买了一个，吃完了很撑。

学习潜水第一天！理论！

第一天的课程是非常基础的，由教练讲述潜水的时候人的特性，空腔变化（耳朵、肺部），可能造成的危害，减压病、肺爆炸、被船的螺旋桨刮残等等，。还讲了水下的压力，bar气压转换，识别潜水装备，BDC、空气瓶、铅块、面镜、蛙鞋、潜水服。还有一些潜水基础知识，印象最深的是：假设地面压强为一个标准大气压，向上1000米为一个大气压（1 bar），向下1米为一个大气压（1 bar），得知我们最后的目标可能是水下15米左右，所以要承受的气压大概为 15bar 当时听见这句话时候，就感觉快窒息了，哈哈。
下午观看一些PADI的潜水教学视频，共四个单元，看完了一半。

晚上的时候去岛上觅食，问了教练，去体验了一下岛上最出名的芒果沙冰，教练和我吐槽到，他一年前来到这里的时候这家店铺的芒果沙冰为100peso（约13元）他每天都吃，现在被游客炒到200peso（约27元），每天吃不起了，哈哈！

卖相虽然不如中国商品精致，但是第一口吸下去口感非常好，凉凉的很浓稠，虽然名字叫做芒果沙冰，但是丝毫感觉不出来有冰的感觉。

学习潜水第二天！平静水域！

第二天的学习地点是在一个最深深度为5米的泳池里面，主要目标是学习20个潜水技巧动作，也是我们第一次穿着潜水服、蛙鞋、面镜、背着空气瓶下水。
穿好这些装备，背好空气瓶的时候，感觉身上的负重大概有40斤左右。。。。😂😂确实很重，但这也是能够保证我们在水下潜水安全和自由的基础。
下水以后 由教练讲述需要做的20多个技巧和动作，大概列了一下教练教学动作以及自己可以做出的动作

水上穿戴及调整装备
水面上BCD充气及泄气
水底呼吸适应
呼吸调节器排水
呼吸调节器寻回
面镜部分进水排水
水中游动
使用备用气源
水下手势
水下穿脱装备
面镜脱下再穿上与排水
在水面口吹气BCD充气与泄气
空气耗尽的练习
水面卸除配重
在水面卸除装备并出水
中性浮力
中性浮力游泳
抽筋解脱
空气耗尽/备用气源混合练习
不断漏气的调节器呼吸
有控制的紧急游泳上升(CESA)
无面镜潜游
潜伴的共生呼吸
在水底卸下并重新穿回水肺装备
在水底卸下并重新穿回配重
在水面卸下并重新穿回水肺装备
在水面卸下并重新穿空对回重系统

想起来学习的时候，下水时空气瓶压力大概为200bar，上水的时候空气瓶压力只有了40bar。
学习和做这些动作，大概使用了5-6个小时的时间呆在泳池里，很累，但是很新鲜，很好玩。
晚上的时候继续观看PADI潜水教学视频。

第二天中午的时候，使用大众点评查询了一些本地商家，看了一些评价，决定体验一下传说中的猪肉汤，一般潜水员上水后都用这个汤来暖身子。

当这碗传说中的猪肉汤拿来后，感觉自己被骗了，这碗价值50人民币的汤，简直没法喝，汤里面有辣椒、茄子、白菜、柿子、大葱以及猪肉，第一次被坑了🙈

没吃饱，又去体验一个香蕉沙冰，感觉香蕉有一些未熟，但是也很棒！国内根本喝不到这样的沙冰，真的很满足。
价格为150peso（19人民币左右）

学习潜水第三天！下海潜水！

第三天是我最期待的下海潜水，经历了第二天的泳池浸泡学习潜水动作，早就期待这一刻了。
由菲律宾本地人开船，将我们在一处浪不大的地方停下。这时我穿戴BCD、空气瓶、蛙鞋、面镜，配重装备等，左脚蛙鞋搭在右脚蛙鞋上，屁股向后挪，后仰式入水。
第一次下到海水里后，一不小心喝了一口海水，当时感觉这海水真TM咸！够劲儿！
快潜到水底的时候，觉得耳朵剧痛感觉快爆炸了，原因是第一次下海没有经验，水底水压过大，人体暴露在外的唯一空腔就是耳朵，太紧张了导致忘记做耳压平衡了。正确方法是每下潜一米左右的时候，做一次耳压平衡，动作为捏住鼻子，向耳朵吹气，鼓起耳膜，达到平衡耳压的目的。
第一次下潜的水底深度大概为9米，在水底抓着绳子稳住做一些第二天在平静水域的动作，比如水底半面镜排水、水底全面镜排水、水底面镜穿戴等记不清楚了。
做潜水动作的时候左右望了一圈周围的生物，简直太美了！
蔚蓝的海底、深黄色的沙子、彩色的珊瑚、黄黑相见的海星、各种各样的鱼🐟，透明的鱼、埋藏在沙子下面的鱼、红色的尼莫，就在你的周围游动，想伸手抓住的时候，又很快的游走了。

做完动作，由教练带领我游了一小圈，最深深度为水下11米，很爽，气压剩余为50bar左右上升回到水面，在水面脱装备，上船。就这样结束了第三天的开放水域学习，将所有动作做完，体验了一圈海底世界，仿佛中了蓝色的毒。

学习潜水第四天！The Last Day! Fun dive!Exam！

由于我们的教练很有责任感，学习时禁止拍照，只希望我们先学会潜水基本动作，所以前三天并没有拍照，和教练商量一下，教练才答应我只在第四天在水下拍照！
第四天只有一上午的时间来体验最后一次潜水，还要将剩下的学习指南表，紧急上升等高级动作做完，在水深深处大概为15米左右的地方游动了一大圈，观看了许多海底生物。急忙的拍了几张在水底的照片，毕竟毕业要紧，哈哈。下次一定补回来。
水底真的太美了！






拍照完，完成了最后一次水下 Fun Dive，开始进行潜水员的期末考试，对于前几天学习的复习，自己还是很有信心，但是考试的时候懵逼了🙈，最后以92分的成绩通过。

为了庆祝自己考试通过，自己特意出去吃了一顿烤牛肉、虾子汤、桔汁大餐，现在虽然看起来卖相一般，但是在菲律宾当地来说，确实非常奢华。。😂😂

最后两天

考试通过后，收到了PADI协会的OW潜水员的临时证件，正式的实体证件听说要等到一个月左右的时间可以收到，所以将邮寄的地址给了教练。

在菲律宾的首都机场附近预定了一家酒店，按照之前来时的路线，原路返回直奔酒店休息。

第二天逛了一下菲律宾最有名的商场 SM MALl OF ASIA 买了一些纪念品，芒果等食物。听说菲律宾的小猪都是吃芒果长大的😂😂😂

深夜的飞机飞回广州，结束旅行。

几天的旅行告一段落，还有一些意犹未尽，总觉得和潜水有缘，水下的世界太美，自己还没玩够，有机会还会继续回来进阶高级潜水员。

很久没写博客了，根本的原因是自己太懒，又觉得生活太平常了，日复一日，一模一样，没什么可写。自己的问题越来越多，进步好慢。

这次的旅行除了学会了潜水之外，让我印象最深刻，感触最深的事情就是英语这门语言。
菲律宾作为日美殖民地，英语作为母语，几乎每个人都会使用英语来交流，当我自己一个人去买东西，订房间，坐出租，买机票，虽然目的都已经达到了，但是沟通起来还是蛮困难，原因有几个:

  1. 他们语速太快/夹杂方言 听不清
  2. 他们语句太长 很多词汇不懂 理解不了
  3. 明白意思 不会说
  4. 会说 但是表达起来很慢 很多关键单词要想

在菲律宾时 看了最近很多新闻 突然发觉英语很重要 真的从未如此想去认真学过英语 背单词
又要有新的目标了 加油
